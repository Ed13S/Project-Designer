System Warnings & Operational Constraints

What Not To Do

  • Do not expose your Gemini API credentials to frontend tracking blocks, analytical aggregators, or third-party storage nodes.
  • Do not refresh or close the session runtime while an active generation pipeline processing sequence is underway.
  • Do not attempt to pass multi-gigabyte text buffers or raw binary assets into the prompt vector arrays.
  • Do not execute local script mutations that bypass the systemic validation checks of the JSON deserializer.

Usage Precautions

  • All generation pipelines pull inference processing units directly from your configured account keys. Monitor your Google AI Studio resource allocations.
  • Local browser context caching utilizes explicit storage references. Clear browser application state manually to hard-reset token maps.
  • The 3D layout matrix serves as a relative structural visualization tool, not a certified structural engineering CAD blueprint.
